Keyboard shortcuts

Important ones

Copy = ctrl + c

Paste = ctrl + v

Cut = ctrl + x

Save = ctrl + s

Undo = ctrl + z

Redo = ctrl + shift + z

Not that important

Reopening closed files, works in the brower as well = ctrl + shift + t

Select all = ctrl + a

Finding in a file = ctrl + f

Finding in the project = ctrl + shift + f

Not that important

Moving lines of code = alt + arrow (up or down)

Going to next line = ctrl + enter

Moving left or right over a whole word = ctrl + arrow (left or right)

Select some text on one line = ctrl + shift + arrow (left or right)

Selecting multiple lines at the same time = ctrl + shift + arrow (up or down)
